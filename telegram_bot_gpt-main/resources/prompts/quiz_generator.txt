Створи 3 унікальних питання для квізу на довільну тему із загальних знань.
Кожне питання повинно мати 4 варіанти відповіді та одну правильну відповідь.
Відповідь має бути повернута **виключно** у форматі JSON.

**ЖОДНИХ ВСТУПНИХ СЛІВ, ПОЯСНЕНЬ ЧИ ДОДАТКОВОГО ТЕКСТУ, ЛИШЕ ЧИСТИЙ JSON.**

Структура JSON повинна бути масивом об'єктів. Кожен об'єкт має мати такі ключі:
- "question" (рядок)
- "options" (масив з 4 рядків)
- "correct_answer" (рядок, який ТОЧНО збігається з одним із варіантів у "options")

Приклад бажаного формату (не включай його у відповідь, використовуй як приклад):
[
  {
    "question": "Яка найменша планета Сонячної системи?",
    "options": ["Марс", "Меркурій", "Венера", "Плутон"],
    "correct_answer": "Меркурій"
  },
  {
    "question": "Хто написав твір 'Захар Беркут'?",
    "options": ["Тарас Шевченко", "Іван Франко", "Леся Українка", "Ольга Кобилянська"],
    "correct_answer": "Іван Франко"
  },
  {
    "question": "Який хімічний елемент має символ 'Fe'?",
    "options": ["Фосфор", "Фтор", "Залізо", "Золото"],
    "correct_answer": "Залізо"
  }
]
