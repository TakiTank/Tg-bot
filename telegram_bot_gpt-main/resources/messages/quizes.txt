ТЕМА 3: ОСНОВИ PYTHON (ПРОГРАМУВАННЯ)

Питання: Як називається структура даних у Python, яка є впорядкованою, незмінною (immutable) та може містити елементи різних типів?
Варіанти: А) List, Б) Tuple, В) Set, Г) Dictionary.
Відповідь: Б) Tuple (кортеж).

Питання: Яке ключове слово використовується для визначення функції в Python?
Варіанти: А) function, Б) func, В) define, Г) def.
Відповідь: Г) def.

Питання: Що виведе наступний код: print(5 // 2)?
Варіанти: А) 2.5, Б) 2, В) 3, Г) Помилка.
Відповідь: Б) 2 (Це оператор цілочисельного ділення).

Питання: Який метод використовується для додавання елемента в кінець списку (list) у Python?
Варіанти: А) insert(), Б) add(), В) append(), Г) extend().
Відповідь: В) append().

Питання: Що означає PEP 8 у контексті Python?
Варіанти: А) Пакет для обробки помилок, Б) Офіційний стандарт стилю коду, В) Пропозиція щодо покращення швидкодії, Г) Тип ліцензії.
Відповідь: Б) Офіційний стандарт стилю коду (Python Enhancement Proposal 8).
